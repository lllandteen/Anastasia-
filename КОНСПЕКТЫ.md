#ЦИКЛЫ FOR

* Цикл for берёт один элемент из последовательности за другим и выполняет блок для каждого элемента
1. python получает последовательность (например, список, строку, range() и тд)
2. Извлекает первый элемент → подставляет его в переменную цикла
3. Выполняет тело цикла
4. Переходит к следующему элементу
5. Повторяет, пока элементы не закончатся.
6. Когда элементы закончились — цикл заканчивается.
```py
# Перебор списка
fruits = ["apple", "banana", "pear"]
for f in fruits:
    print(f)

# Перебор строки
for ch in "hello":
    print(ch)

# Цикл с range()-генератор чисел
for i in range(5):
    print(i)
# Вывод:0 1 2 3 4

# Вложенные циклы
for i in range(3):
    for j in range(2):
        print(i, j)
```

 ```py
else:

 ```
 
 # Цикл while
 * Цикл while-выполняет блок кода пока условие истинно (пока оно равно True)
 Проверяется условие.
 
 Синтаксис
```py
while условие:
    действие
```

Если оно True → выполняется тело цикла.
После выполнения тело цикла снова возвращается к условию.
Когда условие станет False, цикл заканчивается
```py
i = 1
while i <= 5:
    print(i)
    i += 1
```


```py
i = 1
while i <= 5:
    print(i)
```
Такой код будет выводить 1 бесконечно, потому что i не меняется.
# Конструкции Match-case
* match case - Она позволяет удобно проверять значение переменной и выполнять разный код в зависимости от того, чему оно равно.

-Python смотрит значение переменной в match.
Сравнивает его с вариантами case.
Находит совпадение.
Выполняет код внутри подходящего case.
Другие варианты не выполняются.
```py
match переменная:
    case значение1:
        действие
    case значение2:
        действие
    case _:
        действие по умолчанию
```
пример:
```py
color = "red"

match color:
    case "red":
        print("Стоп!")
    case "yellow":
        print("Жди.")
    case "green":
        print("Иди!")
    case _:
        print("Неизвестный цвет")
```
Когда ни одно из условий не совпадает, case _ выполняется автоматически. Например, если значение переменной не соответствует ни одному из явно указанных case, то управление передаётся блоку case _. Это помогает избежать ошибок и предусмотреть обработку непредвиденных значений.
# Условные конструкции if, elif и else
* if - Используется, когда нужно выполнить код только если условие True.
```py
age = 18

if age >= 18:
    print("Совершеннолетний")
```
* if + else - срабатывает, если условие в if — False.
```py
age = 15

if age >= 18:
    print("Совершеннолетний")
else:
    print("Несовершеннолетний")
```
* if + elif + else - elif означает «иначе если».
Используется, когда условий несколько.
```py
grade = 4

if grade == 5:
    print("Отлично")
elif grade == 4:
    print("Хорошо")
elif grade == 3:
    print("Удовлетворительно")
else:
    print("Плохо")
```
принцип работы:

1. Python проверяет условие в if.
2. Если оно True → выполняет этот блок, остальные пропускает.
3. Если False → переходит к elif.
4. Если elif тоже False → идёт дальше.
5. Если ни одно условие не подошло → выполняется else.

Важно, что выполняется только один блок, первый подходящий.

# Коллекции
* Колекции — это структуры данных, которые хранят несколько элементов в одной переменной.
Элементы могут быть числами, строками, списками и даже другими коллекциями.
# Список(list) - изменяемая, упорядоченная коллекция элементов.
1. Создаём список: my_list = [1, 2, 3]
2. Доступ по индексу: my_list[0] → 1
3. Изменяем элемент: my_list[1] = 5
4. Добавляем элемент: my_list.append(6)
5. Удаляем элемент: my_list.remove(5)
```py
# 1. Создание списка
my_list = [1, 2, 3]
print(my_list)  # [1, 2, 3]

# 2. Доступ по индексу
print(my_list[0])  # 1

# 3. Изменение элемента
my_list[1] = 5
print(my_list)  # [1, 5, 3]

# 4. Добавление элемента
my_list.append(6)
print(my_list)  # [1, 5, 3, 6]

# 5. Удаление элемента
my_list.remove(5)
print(my_list)  # [1, 3, 6]
```

# Кортежи(tuple) - Упорядоченная неизменяемая коллекция. После создания элементы нельзя добавить, удалить или изменить
1. Создаём кортеж: point = (10, 20)
2. Доступ по индексу: point[0] → 10

# Словарь(Dict) - коллекция ключ → значение, упорядоченная 
Изменяемая коллекция — набор пар «ключ-значение». Ключи в словаре должны быть уникальными и относиться к неизменяемым типам данных (строки, числа, кортежи), в то время как значения могут быть любыми объектами Python.
1. Создаём словарь: person = {"name": "Alice", "age": 25}
2. Доступ по ключу: person["name"] → "Alice"
3. Изменяем значение: person["age"] = 26
4. Добавляем пару: person["city"] = "NY"
5. Удаляем пару: del person["age"]
```py
# 1. Создание словаря
person = {"name": "Alice", "age": 25}
print(person)  # {'name': 'Alice', 'age': 25}

# 2. Доступ по ключу
print(person["name"])  # "Alice"

# 3. Изменение значения
person["age"] = 26
print(person)  # {'name': 'Alice', 'age': 26}

# 4. Добавление новой пары
person["city"] = "NY"
print(person)  # {'name': 'Alice', 'age': 26, 'city': 'NY'}

# 5. Удаление пары
del person["age"]
print(person)  # {'name': 'Alice', 'city': 'NY'}
```
# Множество (set) - неупорядоченная коллекция уникальных элементов.
Неупорядоченная коллекция уникальных элементов. Сами элементы изменять нельзя, но можно изменять само множество, в которое эти элементы входят, добавляя новые элементы или удаляя ненужные.
1. Создаём множество: numbers = {1, 2, 3}
2. Добавляем элемент: numbers.add(4)
3. Удаляем элемент: numbers.remove(2)
4. Проверяем наличие: 3 in numbers → True

!порядок не сохраняется, дубликаты автоматически удаляются.
```py
# 1. Создание множества
numbers = {1, 2, 3, 3}
print(numbers)  # {1, 2, 3} (дубликат удалён)

# 2. Добавление элемента
numbers.add(4)
print(numbers)  # {1, 2, 3, 4}

# 3. Удаление элемента
numbers.remove(2)
print(numbers)  # {1, 3, 4}

# 4. Проверка наличия элемента
print(3 in numbers)  # True
print(5 in numbers)  # False
```

При импорте модуля Python выполняет весь код в нём.
При импорте пакета Puthon выполняет код в файле пакета _init_.py если он имеется.
Все обьекты определенные в модуле или _init_.py, становятся поступны импортирующему коду

import <модуль/пакет_1> [ as <псевдоним_1> ] [, модуль/пакет_2 as <псевдоним_2> ]...

- После ключевого слова inport указывается название модуля. одной конструкцией можно подключить несколько модулей, но рекомендуется так не делать.
